# 图片素材目录

将你的图片文件放在这个目录下。

## 使用方法

```jsx
import {Img, staticFile} from 'remotion';

<Img
  src={staticFile('/images/your-photo.jpg')}
  style={{width: '100%'}}
/>
```

## 支持的格式

- JPG / JPEG
- PNG
- GIF
- SVG
- WebP
- BMP

## 推荐的图片尺寸

- 全屏背景: 1920x1080 或更大
- 照片: 1920x1080
- 缩略图: 400x300
- 图标: 64x64 或 128x128
- Logo: 500x500
